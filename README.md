# DIED
Welcome to the "Doran Iberian Energy Dashboard" (TM) -- DIED

This is a dashboard showing live and historic data relating to the Iberian electricity system, with a primary focus on Spain. 

To use the dashboard, first check the top right of the page to see if the data is up-to-date. If the data is shown as "CACHED" it will be from the latest scrape (time shown) that has been stored in a .json file. To see live data, you can manually run an API call to ESIOS via the "REFRESH" button in the same panel. 

Once your data is current, feel free to explore. Each module has a rollover info section which explains what that particular module is showing. You can change the chronology from the top left "PERIOD" section. This is available for set periods such as today, yesterday, last 7 days, month to date etc., and there is also a custom function to pick your own time range.

This dashboard is a work in process. To see the current new feature additions, hover over the version number (e.g. V0.4) at the top middle, next to the dashboard title. New features will be committed with time. 

NB: ESIOS data operates on a 1-2hr time lag, meaning this data may be more useful at present for historical comparisons. I shall look to add in live data with time (possibly from Entso-E), as well as features to export data (CSV, XLSX) and do immediate comparisons vs. equivalent period being assessed.
